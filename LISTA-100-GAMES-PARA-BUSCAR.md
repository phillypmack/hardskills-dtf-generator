- ✅ Licença: CC BY 3.0 (uso comercial OK)
- ✅ Download direto em SVG
- ✅ Personalize cores antes de baixar

**Como usar:**
```
1. Acesse https://game-icons.net/
2. Use a busca (ex: "sword", "gun", "minecraft")
3. Clique no ícone desejado
4. Clique em "Download SVG"
5. Salve como: ColaSVG/NomeDoGame.svg
```

## 🚀 Workflow Recomendado

### Passo a Passo:

```bash
# 1. Crie uma pasta temporária para organizar
mkdir game-icons-download

# 2. Baixe os ícones de Game-Icons.net ou outras fontes

# 3. Renomeie os arquivos para nomes descritivos:
#    - fortnite.svg
#    - minecraft.svg (já tem)
#    - gta5.svg
#    - etc.

# 4. Copie todos para a pasta svg/
cp game-icons-download/*.svg "C:/Users/felip/Downloads/Aplicações/Cola/svg/"

# 5. Atualize a lista de SVGs
cd "C:/Users/felip/Downloads/Aplicações/Cola"
npm run update-svg-list

# 6. Inicie o servidor
start-server.bat

# 7. Acesse http://localhost:5173
# 8. Vá em "Modelos Locais" - todos os games estarão lá!
```

## 📝 Dicas de Nomenclatura

Use nomes descritivos e sem espaços:

✅ **BOM:**
- `fortnite.svg`
- `call-of-duty.svg`
- `gta5.svg`
- `league-of-legends.svg`

❌ **EVITE:**
- `Call of Duty Warzone 2.0.svg` (espaços)
- `game1.svg` (não descritivo)
- `FORTNITE.SVG` (maiúsculas)
